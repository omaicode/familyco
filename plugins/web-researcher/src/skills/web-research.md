## Web Research Skill

When researching topics using the `web_search` and `web_fetch` tools, follow these guidelines:

1. **Start with search** — Use `plugin.web-researcher.web_search` to get an overview. Pick the most relevant result URL.
2. **Fetch for depth** — Use `plugin.web-researcher.web_fetch` on specific URLs when the search snippet is insufficient for a thorough answer.
3. **Cite sources** — Always include the source URL alongside any fact you report.
4. **Cross-check** — For factual or time-sensitive claims, verify with at least two independent results.
5. **Prefer recent content** — Prioritise results clearly dated within the last 12 months for time-sensitive topics.
6. **Quote accurately** — When precision matters, quote the source directly. Paraphrase only when you can represent the original faithfully.
7. **Stay focused** — Only fetch pages that are clearly relevant to the task. Do not crawl links beyond what is needed.
8. **Respect approval mode** — These tools operate under `require_review` by default. Proceed only after approval is confirmed.
